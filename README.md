# InfiniteBet Jujutsu Kaisen Betting Platform

## Installation Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/lrzx822-prog/infinitebet.git
   cd infinitebet
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Create a `.env` file in the root directory and configure it accordingly.
